# Contributing

Thanks for improving Agent Ready Coding Loop.

This project is meant to help beginner and intermediate humans get better results from coding agents without losing professional engineering discipline.

## Contribution Principles

Good contributions should:

- make the framework clearer;
- improve safety or verification;
- help agents resume across sessions;
- reduce ambiguity for beginners;
- strengthen rubrics without adding unnecessary complexity.

Avoid contributions that:

- make the framework depend on one proprietary agent;
- remove approval gates;
- weaken the contract or scoreboard;
- encourage coding before SDD approval;
- add jargon without plain-language explanation.

## Useful Contributions

- Better `SDD.md` sections.
- Better contract criteria examples.
- Agent-specific compatibility files.
- Example completed projects.
- Stronger debugging rubrics.
- Better `./check` scoreboard examples.
- Context7/source-check examples for popular stacks.

## Development Workflow

1. Read `AGENTS.md`.
2. Open an issue or discussion for larger changes.
3. Keep changes focused.
4. Update templates when changing the framework.
5. Update README or docs when changing user-facing behavior.

## Style

- Prefer plain English.
- Optimize for beginners without weakening rigor.
- Use short headings.
- Keep rules concrete and testable.
