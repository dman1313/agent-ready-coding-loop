# Acknowledgements

Agent Ready Coding Loop was developed by Dwayne Primeau as an independent framework for helping beginner and intermediate humans work with coding agents through durable specs, contracts, plans, checks, debugging records, and handoff documents.

## Inspiration And Prior Art

Some ideas in this framework were informed by public agentic-coding practices and workflow patterns from:

- Addy Osmani's `agent-skills` repository: https://github.com/addyosmani/agent-skills
- OpenAI Codex best-practice guidance: https://developers.openai.com/codex/learn/best-practices
- OpenAI Codex execution plans guidance: https://developers.openai.com/cookbook/articles/codex_exec_plans
- Anthropic Claude Code best practices: https://www.anthropic.com/engineering/claude-code-best-practices
- Cursor agent best practices: https://cursor.com/blog/agent-best-practices
- Google Gemini CLI Conductor / context-driven development: https://developers.googleblog.com/conductor-introducing-context-driven-development-for-gemini-cli/
- Context7 documentation context tooling: https://context7.com/docs/overview

## Relationship To Prior Work

This project does not claim affiliation with or endorsement by any of the projects or companies above.

The Agent Ready Coding Loop combines and adapts broad ideas common across modern coding-agent workflows, including:

- spec-driven development;
- persistent agent instructions;
- plan-first implementation;
- contract-style success criteria;
- testable scoreboards;
- continuous coding loops;
- skeptical review;
- stop-the-line debugging;
- source-grounded implementation.

The specific packaging, beginner/intermediate user model, artifact set, continuous continuation rubric, and repo-level compatibility structure are part of this framework.
